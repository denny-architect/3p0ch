MinIO AIStor
AI Data Lake
  • Protocol: S3-compatible API
  • Function: Training data storage
  • Integration: PromptObject auto-labeling
  • Workflow: Fight video → MinIO → Label → Train

Centralized repository for fight footage, labeled training data, and model checkpoints

YOLOv8
Object Detection
  • Model: YOLOv8n (nano)
  • Classes: Opponent, Arena, Weapon
  • FPS: 60+ on Jetson Orin Nano
  • Training: Custom dataset from fights

Real-time detection and tracking of opponents, hazards, and arena boundaries

AI Training Pipeline

STEP 01
Fight Recording
FPV camera captures 800TVL combat footage
STEP 02
Upload to MinIO
Raw video stored in AIStor bucket
STEP 03
Auto-Labeling
PromptObject labels opponents & hazards
STEP 04
Model Training
PyTorch fine-tunes YOLOv8 on new data
STEP 05
Deploy to Robot
Updated model loaded to Jetson for next fight

The Learning Loop: Every fight generates new training data. Every loss makes the next iteration smarter.

Development Roadmap

PHASE 01
● In Progress
Manual Combat Bot
Build and prove the mechanical platform. Fight manually via FPV remote control. Capture fight footage and establish the data collection workflow.
  • 2WD drivetrain assembly
  • FPV camera + transmitter
  • Remote control via FlySky
  • 3D-printed PETG-CF chassis
  • Weight optimization (1360g max)
  • Fight data capture pipeline
PHASE 02
Planned
Vision + Tracking
Add Jetson Orin Nano and YOLOv8 for real-time opponent detection and tracking. Still manually controlled, but with visual overlays and telemetry.
  • Jetson Orin Nano integration
  • YOLOv8 object detection
  • Opponent tracking overlay
  • Real-time distance estimation
  • Arena boundary detection
  • Training dataset creation
PHASE 03
Planned
Targeting Assist
Semi-autonomous mode: human pilots, AI suggests attack angles and optimal positioning. Hybrid control with AI-assisted decision-making.
  • Attack angle suggestions
  • Optimal positioning hints
  • Hazard avoidance warnings
  • Predictive tracking
  • Semi-autonomous maneuvers
  • Fight strategy logging
PHASE 04
Future
Full Autonomous
AI takes full control. 3p0ch fights independently, adapting strategy in real-time based on learned behaviors from previous fights. The final evolution.
  • Fully autonomous combat
  • Real-time strategy adaptation
  • Opponent behavior prediction
  • Dynamic weapon targeting
  • Arena hazard navigation
  • Continuous learning loop
