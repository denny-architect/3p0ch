## ü§ñ About

**3p0ch** (pronounced "Epoch") is a 3-pound beetleweight combat robot designed to learn from fight data using onboard AI.

- **Weight Class:** Beetleweight (3 lb / 1360g)
- **AI Compute:** NVIDIA Jetson Orin Nano 8GB
- **Storage:** MinIO AIStor for training data
- **Vision:** YOLOv8 real-time object detection
- **League:** NHRL (National Havoc Robot League)

**The robot learns from its losses.**
